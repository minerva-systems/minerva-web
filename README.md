-   Node.js 16+

### Quick Start

```bash
# Clone the repository
git clone https://github.com/minerva-systems/minerva-web.git
cd minerva-web

# Install dependencies
npm install

# Start development server
npm start

# Build for production
npm run build
```

The app will be available at `http://localhost:3000`
